# Expense Tracker (C++)

A simple console-based Expense Tracker built in C++ to practice Object-Oriented Programming, STL, and File Handling.

This project allows users to record income and expenses, organize transactions by category, set a budget, and store data in a file so records can be loaded again later.

## Features

- Add income transactions  
- Add expense transactions  
- View all transactions  
- Search transactions by category  
- Set and track budget  
- Save transactions to file  
- Load saved transactions from file  
- Menu-driven interface

## Concepts Used

- Classes and Objects  
- Vectors (STL)  
- File Handling  
- Functions  
- Switch Case  
- Basic Data Management Logic

## Project Structure

main.cpp  
transactions.txt
